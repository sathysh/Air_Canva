#🖌️ Air Canva
#Air Canva is a lightweight, web-based design tool for creating posters, graphics, and social media content. Built with React and the Canvas API, it offers a fast, responsive experience perfect for quick and beautiful designs — all without the bloat of heavy design software.

#✨ Features
🎨 Real-time, drag-and-drop canvas editing

⚡ Fast and responsive interface using React

🌐 Web-based — no installation required

🧩 Layer management, image/text insertion, and customization

🆓 Open-source and beginner-friendly

#🚀 Tech Stack
Frontend: React

Graphics: HTML5 Canvas API

Styling: Tailwind CSS (optional)
